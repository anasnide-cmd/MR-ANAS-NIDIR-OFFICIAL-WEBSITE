# MR ANAS NIDIR Official Ecosystem Documentation

Welcome to the central documentation for the MR ANAS NIDIR Official Website and its suite of applications. This ecosystem is built as a high-performance, AI-first platform using Next.js and Firebase.

## 📚 Documentation

Comprehensive documentation and **[User Tutorials](./tutorial.md)** are available here.

- [Tutorial: Getting Started](./tutorial.md)
- [Global Architecture](./global-architecture.md)

## 🌌 Core Architecture

- **Framework**: Next.js 16.1 (App Router)
- **Styling**: Vanilla CSS with CSS Modules, Glassmorphism design system.
- **Backend/DB**: Firebase (Auth, Firestore)
- **AI Stack**: Vercel AI SDK (OpenAI GPT-4o)

## 🛠️ Development & Deployment

- **Local Dev**: `npm run dev`
- **Build**: `npm run build`
- **Deploy**: Vercel for web application hosting, and Firestore database rules/indexes.
